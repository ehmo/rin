# User Experience and Behavior

## Scope
Define how single users, shadow profiles, and business/employee profiles behave from onboarding to daily usage.

## Focus Areas
- Onboarding and identity verification sequence.
- Contact import, dedup, and canonical display rules.
- Circle management with minimum user burden.
- Discovery, ranking explanation, and trust states.
- Dispute and recovery UX for ownership/security events.
